# PilotQA AI

## Vision

PilotQA AI is an AI-powered, open-source Quality Engineering Platform that helps QA teams manage the complete software testing lifecycle with intelligent AI assistance.

## Mission

Reduce the manual effort involved in software testing while improving software quality through AI-driven automation and intelligent recommendations.

## Target Users

- Manual Testers
- Automation Testers

## AI During Execution

When a tester marks a test case as FAILED:

PilotQA AI will:

- Analyze the test execution
- Read the requirement
- Read the expected result
- Read the actual result
- Ask:

"Would you like AI to create a defect?"

If YES:

AI prepares:

- Summary
- Description
- Steps to Reproduce
- Expected Result
- Actual Result
- Severity
- Priority
- Environment
- Browser
- Device

The tester reviews the draft before submitting.

---
